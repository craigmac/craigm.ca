RMTK

RMTK (Records Management Toolkit) started as a CLI Python script I wrote seeking to create a package of common records management functions for reuse. The module included functionas like "find all files that are over N amount of days old and return them". I added a GUI front-end to the library later on. I learned about:

  • Building a reusable library;
  • asynchronous vs. blocking code;
  • GUI widgets of the Tk library;
  • and creating, updating, reading, and deleting files.
